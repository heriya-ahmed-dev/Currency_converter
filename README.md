This Currency Converter is a beginner-friendly web application developed using HTML, CSS, and JavaScript. It enables users to convert amounts between multiple international currencies in real time.

The project helps understand core front-end development concepts such as:

DOM manipulation
Event listeners
Conditional logic
Form handling
Basic currency conversion logic
✨ Features:
Convert between multiple currencies
Simple and clean user interface
Instant conversion on button click
Lightweight and responsive design

This project is ideal for beginners who want to strengthen their JavaScript fundamentals by building real-world mini applications.
